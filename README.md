Directions for pushing code to github repository
1. Once you have made your changes and are ready to push to github repository,

In terminal enter:
git add .

2. After you have entered that line enter: git commit -m "Your commit message"

*** The commit message should be a small message with the summary of changes you made ***

3. Last enter: 
git push origin (BranchName)

*** REPLACE (BranchName) with the name of your branch without the parentheses.***
